# 多层网络不出网渗透

大部分企业中，内网是禁止直接访问互联网的，这样可以阻止很多网络攻击，虽然漏洞可以利用但无法出网。这时想要撕出一个口子需要更多的技巧出网方式。

![禁用互联网演示](https://res.vulab.io/img/4dfec772548f046e83bb6351baeab1f5.png)
